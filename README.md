# Vicold.Pindoudou
拼豆图纸编辑器



## 工具简介

这是一个基于微软 Blazor Hybrid 开发的拼豆图纸设计工具，支持 Windows、Android、iOS 和 macOS 平台。工具的主要功能包括：

- 上传图片并将其转换为像素图
- 自定义像素图的宽度和高度
- 使用默认的 Mard 调色板或创建自定义调色板
- 确保生成的像素图颜色在选定的调色板中

## 快速开始

### 步骤 1：启动工具

运行应用程序，在导航菜单中选择 "拼豆图纸设计工具" 页面。

### 步骤 2：上传图片

点击 "上传图片" 按钮，选择一张你想要转换为拼豆图纸的图片。支持的图片格式包括 JPG、JPEG、PNG 和 GIF。

### 步骤 3：设置像素大小

在 "像素宽度" 和 "像素高度" 输入框中，设置你想要的像素图尺寸。最小尺寸为 1x1，最大尺寸为 100x100。

### 步骤 4：选择调色板

在 "调色板" 下拉菜单中，选择你想要使用的调色板：

- **Mard (默认)**：使用内置的 Mard 调色板，包含 221 种颜色
- **自定义**：使用你自己创建的调色板

### 步骤 5：生成像素图

点击 "生成像素图" 按钮，工具将开始处理图片并生成像素图。生成过程中，按钮会显示 "生成中..." 并禁用，生成完成后会显示成功消息。

## 自定义调色板

### 创建自定义调色板

1. 在 "调色板" 下拉菜单中选择 "自定义"
2. 点击 "添加颜色" 按钮，添加新的颜色
3. 使用颜色选择器调整每个颜色
4. 点击 "删除" 按钮，移除不需要的颜色
5. 点击 "保存调色板" 按钮，保存你的自定义调色板

### 保存和加载自定义调色板

自定义调色板会自动保存到应用程序的 `Data/Palettes/Custom.txt` 文件中，下次启动工具时会自动加载。

## 技术说明

### 颜色匹配算法

工具使用欧几里得距离算法来找到与原始图片颜色最接近的调色板颜色，确保生成的像素图颜色在选定的调色板中。

### 图片处理

由于 Blazor Hybrid 的跨平台特性，工具目前使用简化的图片处理方法。在实际应用中，建议使用 JavaScript interop 或其他跨平台库来实现更复杂的图片处理功能。

## 常见问题

### 问题：生成的像素图与原始图片差异较大

**解答**：这是正常现象，因为像素图的颜色被限制在选定的调色板中，并且尺寸可能与原始图片不同。你可以尝试调整像素大小或使用不同的调色板来获得更好的效果。

### 问题：自定义调色板不保存

**解答**：请确保应用程序有写入文件的权限，并且磁盘空间充足。自定义调色板会保存到 `Data/Palettes/Custom.txt` 文件中。

### 问题：工具无法启动

**解答**：请确保你的设备满足应用程序的系统要求，并且已安装必要的依赖项。如果问题仍然存在，请尝试重新安装应用程序。

## 系统要求

- **Windows**：Windows 10 版本 19041.0 或更高版本
- **Android**：Android 7.0 (API 级别 24) 或更高版本
- **iOS**：iOS 14.2 或更高版本
- **macOS**：macOS 11.0 或更高版本
